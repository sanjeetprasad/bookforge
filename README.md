# BookForge

Your all-in-one book intelligence platform. Upload any book, summarize it, build courses, generate social media content, and read with a Kindle-style visual reader.

## Features
- 🔍 Find books free or cheapest
- 📖 Visual PDF reader with images and diagrams
- ✨ AI-powered summaries
- 🎓 Course builder (1 class to 21 days)
- 📱 Social media content generator
- 📝 Notes and highlights

## Built with
- Claude API (Anthropic)
- OpenAI TTS (voice)
- PDF.js (visual rendering)
- Vanilla HTML/CSS/JS
